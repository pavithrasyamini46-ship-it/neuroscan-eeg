About NeuroScan EEG

A comprehensive, final-year B.Tech project implementing a multidimensional EEG-based neurological disorder diagnosis system with secure storage and role-based access.

Project Overview

NeuroScan EEG is a full-stack web application that integrates advanced signal processing, machine learning, deep learning, and cybersecurity techniques to automate the classification of EEG signals into neurological categories (e.g., Normal, Epileptic).

This system was designed as a Multidimensional Framework because it analyses EEG signals across two analytical dimensions:

  • Time Domain: Mean, Variance, STD, Energy, Entropy, Hjorth Activity/Mobility/Complexity
  • Frequency Domain: FFT-based Delta/Theta/Alpha/Beta band power, Spectral Entropy
