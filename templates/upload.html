How it works
  1. Your CSV is uploaded and AES encrypted immediately.
  2. FIR filter + ICA preprocessing applied to remove noise.
  3. 13 time-domain and frequency-domain features extracted.
  4. Selected ML/DL model classifies the EEG signal.
  5. Encrypted prediction report stored securely.
Expected CSV Format

Each row = one EEG segment. Columns = signal amplitude values. Optional final column = label.

col1,col2,col3,...,col178,label
